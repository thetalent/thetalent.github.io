Find a given 4-sequence of colors. There are 6 possible colors.one color can occur more than 1 time. Pick a color from first row and color the fields of one row. Clicking the question mark evaluates the sequence.
Evaluation maps each element to the sequence. First we try to map equal colors at equal place(green pin). Then equal colors but wrong place(yellow pin). Elements that cant be mapped to equal color create a red pin. The pins are sorted by colors and not by places.
